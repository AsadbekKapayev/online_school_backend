2. Configuring data base

2.1 You should use PostgreSQL, if you haven't postgresql you can download https://www.postgresql.org/download/ 

2.2 You should have a database, if you haven't you can create and memorize the name of db

2.3 You can use tool of idea, you shoul open view -> tool window -> database

<img width="356" alt="image" src="https://user-images.githubusercontent.com/101979162/170858155-5a7ab14f-1412-40ea-8698-2ad49aa76743.png">

2.4 Now you should see database tool, then you should press new button it is like "+"

<img width="248" alt="image" src="https://user-images.githubusercontent.com/101979162/170858238-1b156928-1fd0-4eb9-b449-0e0476311f7e.png">

2.5 After you should see window like on the photo, you should choose new -> Data source -> Postgresql

<img width="400" src="https://user-images.githubusercontent.com/101979162/170858303-5a6215b9-18ff-497a-bf42-52fe1b9a73eb.png">

2.6 Now you should see data source window, you should fill user, password, database inputs, after press the testconnection

<img width="592" alt="image" src="https://user-images.githubusercontent.com/101979162/170858333-1943d48d-26b6-403e-bdb9-2994762ada99.png">

2.7 If you create datasource correctly you should see success message like on the photo, after save the URL

<img width="246" alt="image" src="https://user-images.githubusercontent.com/101979162/170858371-500fb8c7-b656-4cd9-9bad-f03b0c5ef12a.png">

2.8 Now go to the src -> main -> resources package and open application properties

<img width="812" alt="image" src="https://user-images.githubusercontent.com/101979162/170858421-3de3639c-9058-48d1-a76e-03f279268644.png">

2.9 Change the URL, username, password. Use your own info, you should paste URL that you copied some steps ago, and enter username and password

2.10 After open the sql_commands file, it is placed near the application.properties file. Then copy all commands

2.11 Open Database window, press right mouse button for your database and press new -> query console

<img width="400" src="https://user-images.githubusercontent.com/101979162/170858576-e466575b-b2e1-42e2-98ac-ad4a17403b9b.png">

2.12 Now you should see console window, you should paste commands and run the window for all commands

<img width="500" src="https://user-images.githubusercontent.com/101979162/170858628-97d790a3-d3b6-445c-876f-34825f14f760.png">
